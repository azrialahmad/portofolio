# Photography Assets

Drop your photos here organized by type:

## Folder structure
```
photography/
  street/      ← street & candid shots
  cosplay/     ← cosplay portraits
design/        ← UI/graphic design work
```

## Supported formats
jpg, jpeg, png, webp, avif

## How it works
No code changes needed — just drop files in and the site automatically picks them up.
File order in the gallery follows alphabetical order, so prefix with numbers to control order:
  01-jakarta-street.jpg
  02-commuter.jpg
  etc.
